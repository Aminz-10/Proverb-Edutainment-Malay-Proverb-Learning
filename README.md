# 📚 Proverb-Edutainment – Malay Proverb Learning App

**Proverb-Edutainment** is an interactive learning platform designed to help users — especially students — understand and memorize **Malay proverbs (Peribahasa Melayu)** in a fun and engaging way.  
It combines **education** and **entertainment (edutainment)** through interactive quizzes, examples, and real-world context.

---

## ✨ Features

- 📖 **Proverb List Mode** – Browse through a curated collection of Malay proverbs with detailed meanings and example situations  
- 🧠 **Quiz Mode** – Test your understanding with fill-in-the-blank and multiple-choice quizzes  
- 🎯 **Progress Tracker** – View your learning progress and number of correctly answered questions  
- 🔊 **Interactive UI** – Colorful and kid-friendly design for better engagement  
- 🧩 **Two Learning Modes**
  - **Learn Mode:** Explore proverbs with examples  
  - **Quiz Mode:** Answer dynamic questions and earn points  

---

## ⚙️ System Overview

The system aims to:
- Promote learning of traditional Malay proverbs through gamification  
- Encourage self-paced learning via interactive challenges  
- Preserve cultural knowledge in a modern, digital form  
- Provide instant feedback to learners for continuous improvement  

---

## 🏗️ System Architecture

| Layer | Description |
|--------|-------------|
| 🎨 **Frontend** | Mobile application interface for learners (Android Studio) |
| 🔧 **Backend** | Firebase Authentication & Firestore Database for storing user data and quiz results |
| 📦 **Database** | Stores proverb list, examples, user progress, and quiz questions |
